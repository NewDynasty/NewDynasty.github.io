切换eclipse package explorer字体大小

Eclipse
  • Eclipse

1.找到eclipse\plugins\org.eclipse.ui.themes_1.1.1.v20151026-1355\css

2.找到对应主题的css，当时是用的Dark theme，所以对应的修改e4-dark-win.css文件，

3.找到如下属性，并在其添加font-size: 10px;
CTabFolder Tree, CTabFolder Canvas {
background-color: #2F2F2F;
color: #CCC;
}
CTabFolder Tree, CTabFolder Canvas {
background-color: #2F2F2F;
color: #CCC;
font-size: 10px;
}
